## Maven
```
<dependencies>
    <dependency> <!-- Derkutils -->
        <groupId>xyz.derkades</groupId>
        <artifactId>Derkutils</artifactId>
        <version>enter version here</version>
    </dependency>
</dependencies>
```
Clone the GitHub repository and `mvn install` it first.
